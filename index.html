MoviesManager Curses

Manage your movies, play them, the TTY way!

Download as .zip Download as .tar.gz View on GitHub

What is it?

MM Curses is a project aiming at bringing a full movie manager to the CLI, usable entirely without X. It is targeted at GUI-haters, Curses-lovers or just movie-lovers who want to try a new way to enjoy their favorite films. It uses mplayer as a backend by default, but you will be able to personalize it as much as you want in the future!

How to help?

You can clone the repository with a simple

$ git clone git://github.com/simsor/moviesmanager-curses.git

Feel free to send bug reports, pull requests or to fork the project if you're interested!
